# Vexra

Look up any Roblox profile instantly.

---

## What it does

Enter a Roblox username and get a full breakdown of that account — profile info, stats, friends, groups, and more. Clean, fast, no login needed.

---

## Features

- Profile overview — avatar, display name, bio, status
- Account stats — friends, groups, account age
- Full details — join date, last location, verified & ban status
- Friends list with avatars
- Groups & roles
